## DFS - 深度优先搜索

```
    0   1   2   3   
0   我
1           *
2       *
3           *   她    

            (0,0)
        下上        
    (0,1)           (1,0)
下上     下上
(0,2)   (1,1)   (1,1)   (2,0)


```

树形结构遍历 相似 递归 


### 总结

深度优先遍历以递归的程序形态 外在表现

## BFS 广度优先搜索

与队列相关

1、2、3层 依次放入队列中遍历

```
    0   1   2   3   
0           我
1      我我  *
2   我    *
3            *  她

            (0,0)                   1层
        
    (0,1)           (1,0)           2层

(0,2)   (1,1)   (1,1)   (2,0)       3层


(0,0)   (0,1) , (1,0)     (0,2) , (1,1) , (1,1) , (2,0)

```
按层级进行处理

(1,1)   (1,1) 冗余 状态舍弃 --> 剪枝


### 广度遍历--解决问题

最优化问题
